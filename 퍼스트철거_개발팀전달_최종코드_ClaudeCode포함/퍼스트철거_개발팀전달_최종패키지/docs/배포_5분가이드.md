# 외부 시연 링크 만드는 5분 가이드
1. 이 폴더를 GitHub private repository에 push
2. Vercel에서 Add New Project → 저장소 선택
3. Framework Preset: Next.js
4. 환경변수는 `.env.example` 기준으로 등록
5. Deploy 클릭
6. Vercel Domains에 `demo.firstdemolition.co.kr` 추가
7. 도메인 DNS에서 Vercel이 안내한 CNAME/A 레코드 등록
8. HTTPS가 활성화되면 카카오톡/문자로 링크 공유

운영 공개 전에는 개인정보 동의문, 사업자정보, 이용약관, 개인정보처리방침, 실제 상담 연락처를 확정한다.
